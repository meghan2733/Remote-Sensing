# Remote-Sensing
This repository contains some project work in field of Remote Sensing.

# Earthquake Analysis
This project was carried out for Southern Alaska. Earthquake measurements (Richter scale) 
from obsevation points were used to carry out Spatial Analysis to nearby areas.

# Potholes Identification
This project used images captured by Drone to spatially analyzing potholes. This project used 
OpenCV library and Python for processing images and feature extraction.
